# Luxury AI Web Design Agency 💎

![Luxury Agency Banner](./public/banner.png)

A high-end, premium landing page built with **Next.js 15**, **React**, and **Tailwind CSS**. Designed specifically for high-ticket coaches, consultants, and premium service providers.

## ✨ Key Features

- **Luxury Aesthetics**: Sophisticated dark mode with glassmorphism and gold accents.
- **AI-Enhanced Workflow**: Showcasing a bespoke design process powered by artificial intelligence.
- **Conversion Focused**: Strategically placed CTA (Call to Action) sections to turn visitors into clients.
- **Smooth Animations**: Interactive UI elements using `framer-motion` (motion/react).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Modern Tech Stack**: Built with the latest Next.js 15 App Router and TypeScript.
- **SEO Optimized**: Pre-configured meta tags, sitemaps, and robots.txt for maximum visibility.

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abhi478jeetur-rgb/luxury-web-design-agency.git
   cd luxury-web-design-agency
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `app/`: Next.js App Router pages and global styles.
- `components/`: Reusable UI components (Navbar, Hero, About, FAQ, etc.).
- `public/`: Static assets like images and videos.
- `hooks/`: Custom React hooks.
- `lib/`: Utility functions.
